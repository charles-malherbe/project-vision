
# Détection de logo sur image

Détection de logo sur image statique avec algorithme SIFT et ORB.


## Prérequis

#### Tkinter

``pip install tk``

#### Pillow

``pip install pillow``

#### MatPlotLib

``pip install matplotlib``

#### OpenCV

``pip install opencv-python``
